# API Testing with REST Assured

## Project Objective

This project focuses on automating API testing using REST Assured, a Java-based framework designed for testing RESTful web services. The project aims to validate the functionality, performance, and security of APIs by automating test cases for CRUD operations (Create, Read, Update, Delete). It integrates with JUnit/TestNG for structured test execution and reporting.

---

## Project Descriprition 

- Initialize the Java project with Maven for dependency management  
- Add REST Assured dependencies  

- Write test cases for:
  - GET  
  - POST  
  - PUT  
  - DELETE requests  

- Validate:
  - Status codes  
  - Response bodies  
  - Headers  
  - JSON schema  

- Generate report with Allure Reports  
- Verify the tests locally using Maven:
  
- Access the API [here](https://jsonplaceholder.typicode.com/)
